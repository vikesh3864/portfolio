## Quick Start

```bash
# Clone the repository
git clone https://github.com/vikesh3864/portfolio.git
cd portfolio

# Install dependencies
pip install django

# Run migrations
python manage.py migrate

# Create admin user
python manage.py createsuperuser

# Start server
python manage.py runserver
```

Visit `http://127.0.0.1:8000` to see the website.

## Admin Panel

Access the admin panel at `/admin` to view contact form submissions.
